# AI Liquid Reality（AI 液态现实）

> **由流体、光影与粒子构成的新一代 AI 交互界面。**

AI Liquid Reality 是一个基于 **Three.js + WebGL Shader** 构建的实验性交互网页，灵感来源于现代 AI 产品设计、液态玻璃（Liquid Glass）视觉语言以及沉浸式空间计算界面。

项目融合了流体背景、动态粒子、毛玻璃（Glassmorphism）和实时光影效果，打造具有未来科技感的 AI 首页体验。

---

## ✨ 项目特色

* 🌊 流体 Shader 动态背景
* ✨ 漂浮粒子空间效果
* 💎 毛玻璃（Glassmorphism）UI
* 🚀 Three.js 实时渲染
* 🎨 液态光效与渐变色彩
* 📱 自适应响应式布局
* 🔐 毛玻璃风格登录弹窗
* ⚡ 轻量、高性能、易于扩展
